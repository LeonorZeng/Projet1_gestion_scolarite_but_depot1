 Gestion de Scolarité - BUT

## 📋 Description
Application de gestion de scolarité permettant de gérer l'inscription des étudiants, 
l'enregistrement de leurs notes et la validation de leur passage à l'année suivante.

## 🎯 Fonctionnalités
- ✅ Inscription des étudiants
- ✅ Enregistrement des notes par cours
- ✅ Vérification automatique de l'admissibilité (notes suffisantes pour progresser)
- ✅ Gestion des données persistantes

## 🛠️ Technologies
- **Langage** : C
- **Concepts** : Structures de données, fichiers, algorithmes de validation

## 📦 Installation & Utilisation
```bash
gcc -o gestion_scolarite main.c
./gestion_scolarite
```
